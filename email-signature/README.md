# Veytra E-Mail-Signatur

## Einrichten

1. `christoph-pfad.html` im Browser öffnen.
2. Die sichtbare Signatur vollständig markieren und kopieren.
3. In Gmail, Outlook oder Apple Mail in das Signaturfeld einfügen.
4. Eine Testmail an dich selbst senden und Desktop- sowie Mobilansicht prüfen.

Die HTML-Version nutzt nur Tabellen und Inline-Styles, damit sie in gängigen E-Mail-Clients stabil dargestellt wird. Das Veytra-Wordmark besteht bewusst aus Text: So sind keine extern geladenen oder geblockten Bilder nötig.

`christoph-pfad.txt` ist die passende Nur-Text-Version.
